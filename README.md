## Technologies
Project is created with:
* HTML
* SCSS
* TypeScript
* React.js
* Firebase (Firestore DB)

* Realtime database saving
* Using local storage to store user data

	
## Setup
To run this project, install it locally using npm:

```
$ cd lights-out
$ npm install
$ npm start
```
